Introduction
============

QPov converts Quake demos into POV-Ray files, ready to render.

Installing
==========
```
$ mkdir go
$ cd go
$ GOPATH=$(pwd) go get github.com/ThomasHabets/qpov
$ GOPATH=$(pwd) go build github.com/ThomasHabets/qpov/cmd/{bsp,dem,mdl,pak,render}
```

=======
